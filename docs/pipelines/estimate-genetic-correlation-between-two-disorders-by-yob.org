This is a guide for using the pre-assembled pipeline that estimates genetic
correlation between two disorders stratified by birth year, using TTE (time-to-event) data.

All code shown in this guide have been written to [[./estimate-genetic-correlation-between-two-disorders-by-yob.R][this R file]], so that you don't have
to copy all code snippets manually.

This pipeline is an extension of the [[./estimate-genetic-correlation-between-two-disorders.org]["estimate genetic correlation between two disorders"]].
We will extend the research question from that pipeline to be:
"What is the genetic correlation between schizophrenia and coronary artery disease
in Danish millenials and their full siblings (people born from 1981 to 1996)
stratified by birth year?"

The same naming convention will be used as the previous pipeline:

- Disorders:
  - ~d1~ = Schizophrenia
  - ~d2~ = Coronary Artery Disease
- Cohorts:
  - ~c1~ = Target cohort
  - ~c2~ = Individuals with at least 1 diagnosed relative of disorder 1
  - ~c3~ = Individuals with at least 1 diagnosed relative of disorder 2
- Results:
  - ~re_d1_c1~ = Risk estimates of disorder 1, cohort 1
  - ~re_d1_c2~ = Risk estimates of disorder 1, cohort 2
  - ~re_d1_c3~ = Risk estimates of disorder 1, cohort 3
  - ~re_d2_c1~ = Risk estimates of disorder 2, cohort 1
  - ~re_d2_c3~ = Risk estimates of disorder 2, cohort 3
  - ~h2_d1~ = Heritability of disorder 1 (cohort 1 vs cohort 2)
  - ~h2_d2~ = Heritability of disorder 2 (cohort 1 vs cohort 3)
  - ~gc_d1_d2~ = Genetic correlation (disorder 1 vs disorder 2)

The data derivation diagram is exactly the same as the previous pipeline:

[[file:../diagrams/gc-dag.png]]

The answer to the research question above will be in result ~gc_d1_d2~.

#+BEGIN_SRC R :exports code :results output
run_cif_stratified <- function(disorder, cohort) {
  onset <- get(paste0(disorder, "_earliest_onset_age"))
  tte   <- get(paste0(cohort, "_tte")) |>
    rename(
      failure_status = !!as.name(paste0(disorder, "_failure_status")),
      failure_time   = !!as.name(paste0(disorder, "_failure_time"))
    )

  cif_analysis$run(
    tte = tte,
    earliest_onset = onset,
    group_columns = list("born_at_year")
  )
}

re_d1_c1 <- run_cif_stratified("d1", "c1")
re_d1_c2 <- run_cif_stratified("d1", "c2")
re_d1_c3 <- run_cif_stratified("d1", "c3")
re_d2_c1 <- run_cif_stratified("d2", "c1")
re_d2_c3 <- run_cif_stratified("d2", "c3")
#+END_SRC

run_h2_stratified <- function(c1_estimates, c2_estimates) {
  combined_estimates <- c1_estimates |>
    inner_join(c2_estimates, by = join_by(time, born_at_year)) |>
    rename(
      cohort1_estimates = estimate.x,
      cohort1_cases     = cases.x,
      cohort2_estimates = estimate.y,
      cohort2_cases     = cases.y
    ) |>
    select(time, born_at_year, starts_with("cohort"))

  h2_analysis$run(
    relationship_kind = relationship_kind,
    estimates         = combined_estimates
  )
}
