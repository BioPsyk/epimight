#+TITLE: Cumulative incidence
#+AUTHOR: Richard Zetterberg <richard.zetterberg@regionh.dk>

Cumulative incidence is produced by the [[https://r6.r-lib.org/articles/Introduction.html][R6 class]] ~CumulativeIncidenceAnalysis~ included in this package.
You can learn how to use this class, either by reading the reference documentation below or read
[[./guide.org][this guide that shows you how it is used, step-by-step with code examples.]]

* Functional reference

To use the class, you need to create an instance first:

#+BEGIN_SRC R
analysis <- CumulativeIncidenceAnalysis$new()
#+END_SRC

Then you use that instance to run one of the 3 functions it exposes:

- run :: The main function that produces the estimates from TTE data
- get_group_values :: Helper function that retrieves all distinct values of a TTE data column
- make_group_summary :: Helper function that summarizes stratified results for each group

** Using function ~run~

This function accepts 4 named arguments:

- tte (required) :: A ~data.table~ that contains the TTE data.
- group_columns (optional) :: A ~list~ that contains names of the columns to stratify the results on.
- earliest_onset (optional, defaults to 1) :: Positive integer that represents earliest age of onset to allow. Result rows with a ~time~ value less than this value will be removed.
- latest_onset (optional, defaults to 100) :: Positive integer that represents latest age of onset to allow. Result rows with a ~time~ value larger than this value will be removed.

*** Non-stratified usage

By only supplying the TTE data, without any other arguments, the results are not stratified.
Here's an example where we read a single dataset that contains TTE data for schizophrenia,
convert it to a ~data.table~ and then input into the run function:

#+BEGIN_SRC R
estimates <- analysis$run(
  tte = read_csv("./tmp/data/tte_SCZ.csv") |> as.data.table()
)
#+END_SRC

*** Stratified usage

In order to stratify the results, we need to supply which column(s) contains the grouping
values. Multiple columns can be used, if you want to create groups for each combination of
values from the columns.

Here's an example where we want to stratify the results on birthyear:
We

Here's an example where we read a single dataset that contains TTE data for schizophrenia,
that contains an extra column called ~born_at_year~ which contains the year the sample was
born. We then use the run function and also tell it to stratify the results on the ~born_at_year~
column:

#+BEGIN_SRC R
estimates <- analysis$run(
  tte           = read_csv("./tmp/data/tte_SCZ.csv") |> as.data.table(),
  group_columns = list("born_at_year")
)
#+END_SRC
