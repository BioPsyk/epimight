This is a guide for using the pre-assembled pipeline that estimates genetic
correlation between two disorders stratified by birth year, using TTE (time-to-event) data.

All code shown in this guide have been written to [[./guide-yob.R][this R file]], so that you don't have
to copy all code snippets manually.

This pipeline is an extension of the [[./guide.org]["estimate genetic correlation between two disorders"]].
We will extend the research question from that pipeline to be:
"What is the genetic correlation between schizophrenia and coronary artery disease
in Danish millenials and their full siblings (people born from 1981 to 1996)
stratified by birth year?"

The same naming convention will be used as the previous pipeline:

- Disorders:
  - ~d1~ = Schizophrenia
  - ~d2~ = Coronary Artery Disease
- Cohorts:
  - ~c1~ = Target cohort
  - ~c2~ = Individuals with at least 1 diagnosed relative of disorder 1
  - ~c3~ = Individuals with at least 1 diagnosed relative of disorder 2
- Results:
  - ~cif_d1_c1~ = Risk estimates of disorder 1, cohort 1
  - ~cif_d1_c2~ = Risk estimates of disorder 1, cohort 2
  - ~cif_d1_c3~ = Risk estimates of disorder 1, cohort 3
  - ~cif_d2_c1~ = Risk estimates of disorder 2, cohort 1
  - ~cif_d2_c3~ = Risk estimates of disorder 2, cohort 3
  - ~h2_d1~ = Heritability of disorder 1 (cohort 1 vs cohort 2)
  - ~h2_d2~ = Heritability of disorder 2 (cohort 1 vs cohort 3)
  - ~gc_d1_d2~ = Genetic correlation (disorder 1 vs disorder 2)

The data derivation diagram is exactly the same as the previous pipeline:

[[file:../diagrams/gc-dag.png]]

The answer to the research question above will be in result ~gc_d1_d2~.

Like the non-stratified pipeline, we need to create the 5 risk estimates:

- cif_d1_c1 :: Risk estimates of disorder 1, cohort 1
- cif_d1_c2 :: Risk estimates of disorder 1, cohort 2

- cif_d2_c1 :: Risk estimates of disorder 2, cohort 1

#+BEGIN_SRC R :exports code :results output
run_cif_stratified <- function(disorder, cohort) {
  onset <- get(paste0(disorder, "_earliest_onset_age"))
  tte   <- get(paste0(cohort, "_tte")) |>
    rename(
      failure_status = !!as.name(paste0(disorder, "_failure_status")),
      failure_time   = !!as.name(paste0(disorder, "_failure_time"))
    )

  cif_analysis$run(
    tte = tte,
    earliest_onset = onset,
    stratify_columns = list("born_at_year")
  )
}

cif_d1_c1 <- run_cif_stratified("d1", "c1")
cif_d1_c2 <- run_cif_stratified("d1", "c2")
cif_d1_c3 <- run_cif_stratified("d1", "c3")
cif_d2_c1 <- run_cif_stratified("d2", "c1")
cif_d2_c3 <- run_cif_stratified("d2", "c3")
#+END_SRC

- h_d1 :: Heritability of disorder 1 (cohort 1 vs cohort 2)

run_h2_stratified <- function(c1_estimates, c2_estimates) {
  combined_estimates <- c1_estimates |>
    inner_join(c2_estimates, by = join_by(time, born_at_year)) |>
    rename(
      c1_cif       = cif.x,
      c1_cif_cases = cases.x,
      c2_cif       = cif.y,
      c2_cif_cases = cases.y
    ) |>
    select(time, born_at_year, starts_with(c("c1", "c2")))

  h2_analysis$run(
    relationship_kind = relationship_kind,
    estimates         = combined_estimates
  ) |> select(time, born_at_year, h2, se)
}

Now that the risk and heritability estimates have been produced, the only thing left is to
produce a table that contains all the risk estimates and heritabilities, which finally
can be passed to the genetic correlation analysis:

#+BEGIN_SRC R :exports both :results output
re_combined <- inner_join(cif_d1_c1, cif_d1_c3, by = join_by(time, born_at_year)) |>
  rename(
    d1_c1_cif       = cif.x,
    d1_c1_cif_cases = cases.x,
    d1_c3_cif       = cif.y,
    d1_c3_cif_cases = cases.y
  ) |>
  select(time, born_at_year, starts_with("d1_")) |>
  inner_join(cif_d2_c1, by = join_by(time, born_at_year)) |>
  rename(
    d2_c1_cif       = cif,
    d2_c1_cif_cases = cases
  ) |>
  select(time, born_at_year, starts_with("d1_"), starts_with("d2_"))

# Join both h2 estimates into a single estimate
h2_combined <- inner_join(h2_d1, h2_d2, by = join_by(time, born_at_year)) |>
  rename(
    d1_h2 = h2.x,
    d2_h2 = h2.y
  ) |>
  select(time, born_at_year, starts_with("d1_"), starts_with("d2_"))

# Join cumulative incidence and heritability estimates into a single dataset
combined <- inner_join(re_combined, h2_combined, by = join_by(time, born_at_year)) |>
  group_by(born_at_year) |>
  arrange(desc(time)) |>
  filter(row_number() == 1) |>
  arrange(born_at_year) |>
  as.data.table()
