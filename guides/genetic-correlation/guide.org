This is a guide for generating genetic correlation between two disorders using TTE data.

All code shown in this guide have been written to [[./guide.R][this R file]], so that you don't have
to copy all code snippets manually.

This guide is similar to the [[../heritability/guide.org]["estimate heritability of disorder"]]-guide.
We will extend the research question from that guide to be:
"What is the genetic correlation between schizophrenia and coronary artery disease
in Danish millenials and their full siblings (people born from 1981 to 1996)?"

To avoid confusion the following naming convention will be used:

- Disorders:
  - ~d1~ = Schizophrenia
  - ~d2~ = Coronary Artery Disease
- Cohorts:
  - ~c1~ = Target cohort
  - ~c2~ = Individuals with at least 1 diagnosed relative of disorder 1
  - ~c3~ = Individuals with at least 1 diagnosed relative of disorder 2
- Results:
  - ~cif_d1_c1~ = Risk estimates of disorder 1, cohort 1
  - ~cif_d1_c2~ = Risk estimates of disorder 1, cohort 2
  - ~cif_d1_c3~ = Risk estimates of disorder 1, cohort 3
  - ~cif_d2_c1~ = Risk estimates of disorder 2, cohort 1
  - ~cif_d2_c3~ = Risk estimates of disorder 2, cohort 3
  - ~h_d1~ = Heritability of disorder 1 (cohort 1 vs cohort 2)
  - ~h_d2~ = Heritability of disorder 2 (cohort 1 vs cohort 3)
  - ~gc_d1_d2~ = Genetic correlation (disorder 1 vs disorder 2)

* Data processing

Now that we have defined all arguments to use, it's time to read and process the
TTE data that we are going to analyze. The TTE data needs to fulfill the same
requirements as the [[../heritability/guide-yob.org][previous guide]], but in this case we will have one TTE dataset
for disorder 1 and one TTE dataset for disorder 2.

The example data in this guide is contained in two files called:

- ~tte_SCZ_FS.csv~
- ~tte_CAD_FS.csv~

Like before, we simply read these files and convert them into a ~data.table~:

#+BEGIN_SRC R :eval no :exports code
d1_tte <- read_csv("./tte_SCZ_FS.csv") |> as.data.table()

** Risk estimates

Like earlier guides, we will use the class ~CumulativeIncidenceAnalysis~
to produce the estimates:

#+BEGIN_SRC R :exports code :results output
cif_analysis <- CumulativeIncidenceAnalysis$new()
#+END_SRC

#+RESULTS:

With the analysis class and the survival data for the 3 cohorts, we can produce
the 5 risk estimates:

- cif_d1_c1 :: Risk estimates of disorder 1, cohort 1
- cif_d1_c2 :: Risk estimates of disorder 1, cohort 2
- cif_d1_c3 :: Risk estimates of disorder 1, cohort 3
- cif_d2_c1 :: Risk estimates of disorder 2, cohort 1
- cif_d2_c3 :: Risk estimates of disorder 2, cohort 3

Using some metaprogramming, we can create a function that
accepts the disorder and cohort according to the naming convention,
that when run will execute the risk estimate analysis using the correct
inputs:

#+BEGIN_SRC R :exports code :results output
run_cif_prefixed <- function(disorder, cohort) {
  onset <- get(paste0(disorder, "_earliest_onset_age"))
  tte   <- get(paste0(cohort, "_tte")) |>
    rename(
      failure_status = !!as.name(paste0(disorder, "_failure_status")),
      failure_time   = !!as.name(paste0(disorder, "_failure_time"))
    )

  cif_analysis$run(tte = tte, earliest_onset = onset)
}

cif_d1_c1 <- run_cif_prefixed("d1", "c1")
cif_d1_c2 <- run_cif_prefixed("d1", "c2")
cif_d1_c3 <- run_cif_prefixed("d1", "c3")
cif_d2_c1 <- run_cif_prefixed("d2", "c1")
cif_d2_c3 <- run_cif_prefixed("d2", "c3")
#+END_SRC

Like earlier heritability guides, we need to combine the last risk estimates from the two

#+BEGIN_SRC R :exports code :results output
run_h2 <- function(c1_estimates, c2_estimates) {
  combined_estimates <- c1_estimates |>
    inner_join(c2_estimates, by = join_by(time)) |>
    rename(
      c1_cif       = cif.x,
      c1_cif_cases = cases.x,
      c2_cif       = cif.y,
      c2_cif_cases = cases.y
    ) |>
    select(time, c1_cif, c1_cif_cases, c2_cif, c2_cif_cases)

  h2_analysis$run(
    relationship_kind = relationship_kind,
    estimates         = combined_estimates
  ) |> select(time, h2, se, l95, u95)
}

h2_d1 <- run_h2(cif_d1_c1, cif_d1_c2)
h2_d2 <- run_h2(cif_d2_c1, cif_d2_c3)

** Genetic correlation

Now that the risk and heritability estimates have been produced, the only thing left to do
is to produce the genetic correlation:

#+BEGIN_SRC R :exports both :results output
# Join last cumulative incidence estimates into a single dataset
re_combined <- inner_join(cif_d1_c1, cif_d1_c3, by = join_by(time)) |>
  rename(
    d1_c1_cif       = cif.x,
    d1_c1_cif_cases = cases.x,
    d1_c3_cif       = cif.y,
    d1_c3_cif_cases = cases.y,
  ) |>
  select(time, starts_with("d1_")) |>
  inner_join(cif_d2_c1, by = join_by(time)) |>
  rename(
    d2_c1_cif       = cif,
    d2_c1_cif_cases = cases
  ) |>
  select(time, starts_with("d1_"), starts_with("d2_"))

# Join both h2 estimates into a single estimate
h2_combined <- inner_join(h2_d1, h2_d2, by = join_by(time)) |>
  rename(
    d1_h2 = h2.x,
    d2_h2 = h2.y
  ) |>
  select(time, starts_with("d1_"), starts_with("d2_"))

# Join cumulative incidence and heritability estimates into a single dataset
combined <- inner_join(re_combined, h2_combined, by = join_by(time)) |>
  arrange(desc(time)) |>
  filter(row_number() == 1) |>
  as.data.table()
#+END_SRC
